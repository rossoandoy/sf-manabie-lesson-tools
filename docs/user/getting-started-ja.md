# 利用開始ガイド（教室長 / 事務担当者向け）

Manabie Lesson Tools を **初めて使う** ときの手順です。  
日常操作の詳細は [操作マニュアル](./operator-manual-ja.md) を参照してください。

**対象バージョン:** v0.3.0  
**対象環境:** Salesforce Sandbox（trg2--extuat）

---

## 事前準備

- [ ] **Google Chrome** ブラウザ
- [ ] Salesforce **Sandbox** へのログイン権限
- [ ] 配布された **`manabie-lesson-tools-v0.3.0.zip`**
- [ ] 所属校舎（Account）が Salesforce 上で設定済み（通常は Affiliation で自動）

---

## Step 1: 拡張機能をインストール

1. 受け取った ZIP を **解凍** する  
   - 解凍フォルダ内に `manifest.json` があることを確認
2. Chrome アドレスバーに `chrome://extensions` と入力して Enter
3. 右上 **デベロッパーモード** を **ON**
4. **パッケージ化されていない拡張機能を読み込む** をクリック
5. 解凍したフォルダを選択
6. 一覧に **Manabie Lesson Tools** が表示され、エラーがないことを確認

```
chrome://extensions
  └─ Manabie Lesson Tools  [ON]
       バージョン 0.3.0
```

---

## Step 2: Salesforce にログイン

1. Chrome で **trg2--extuat** Sandbox を開き、通常どおりログイン
2. ログイン状態を保ったまま作業を続ける（別ブラウザやシークレットウィンドウでは拡張が連携できません）

---

## Step 3: ダッシュボードを開く

1. ブラウザ右上の **Manabie Lesson Tools** アイコン（パズルピースからピン留め推奨）をクリック
2. Popup で接続状態を確認
3. **ダッシュボードを開く** をクリック
4. 新しいタブにダッシュボードが開き、ヘッダーに **Sandbox** バッジと org URL が表示される

---

## Step 4: 初回セットアップ（必須）

画面上部またはチェックリストに従い、次を **順番に** 実行します。

### 4-1. 前提マスタ同期

1. ヘッダーの **前提マスタ同期** をクリック
2. 成功 toast「前提マスタ同期が完了しました」が出ること
3. エラーが出た場合 → [トラブルシューティング](#トラブルシューティング) へ

### 4-2. 所属校舎・ブース数の確認

1. **コマ組** タブを開く
2. 左 **表示設定** で次を確認:
   - **拠点（Account）** … 自分の教室になっている
   - **ブース数** … Salesforce の Account と一致（**変更不可**）
3. 自動設定されない場合:
   - チェックリストのヒントを読む
   - 拠点を手動選択し「手動で変更」トグルを ON

### 4-3. （任意）請求データ同期

回数報告で請求列を使う場合のみ:

1. **回数報告** タブ → **請求データ同期（F13）**

---

## Step 5: 日常業務の入口

| やりたいこと | 開くタブ |
|--------------|----------|
| 週のコマを入力・編集 | **コマ組** |
| 一覧で確認・SF 同期 | **授業一覧** |
| 月次回数・署名用印刷 | **回数報告** |
| 休校日を登録 | **休校日** |
| SF へ一括送信 | **Manabie登録** または 授業一覧の Sync Dock |

### コマ組の基本操作（最小）

1. **ブース表** で日付×ブース×時限のセルを操作
2. **講師 / 生徒** … セルをダブルクリック、または **▾ ボタン**（授業一覧・回数報告の名前欄）で検索モーダルから選択
3. **教科** … プルダウンから選択
4. 入力後、**授業一覧** の Sync Dock から SF へ同期（F19 等）

---

## トラブルシューティング

| 症状 | 対処 |
|------|------|
| 「セッションなし」「Session expired」 | Salesforce Sandbox に再ログイン → ダッシュボードを開き直す |
| 前提マスタ同期でエラー | IT 担当に連絡（org 設定の問題の可能性あり） |
| ブース数が合わない | **前提マスタ同期** を再実行。それでも合わない場合は IT 担当へ |
| 生徒・講師の一覧が空 | 拠点（Account）が正しいか確認。Affiliation 設定を IT に確認依頼 |
| Manabieへ登録が押せない | 前提マスタ同期済みか、入力データに error がないか確認 |
| Production org を開いている | **Sandbox** バッジが付いた URL で作業する |
